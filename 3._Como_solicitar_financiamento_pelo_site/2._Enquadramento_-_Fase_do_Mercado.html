2. Enquadramento - Fase do Mercado

Created Wednesday 21 May 2014 - Atualizado em 18/01/2016

ENQUADRAMENTO DA PROPOSTA DE MPE PELA ÁREA DE MERCADO

O enquadramento de propostas é realizado pelos técnicos que atuam na área de Mercado.

1º PASSO - Validar dados da empresa

- Verificar se E-mail e fone cadastrados estão corretos;
- Validar se Empresa está faturando há mais de 12 meses;
- Se a empresa macou opção que ainda "NÃO ESTÁ FATURANDO", verificar no menu ANEXOS CLIENTE se o plano de negócios foi incluido.
- Validar se o ramo de atuação da proponente é passível de apoio;
- Verificar se a finalidade do recurso realmente é para atividade que esteja contida em algum dos CNAEs da empresa ou é para uma atividade que a empresa pretende passar a explorar.

2º PASSO - Validar Parâmetros da Proposta

- Verificar se o faturamento informado corresponde à realidade. Caso haja valores muito arredondados validar com o cliente.
- Verificar se a descrição do projeto bate com os itens de financiamento.
- Verificar se os itens de financiamento correspondem às linhas que operacionalizamos.
- Verificar a opção de Garantia assinalada pelo cliente.
- Verificar se prazo e carência estão de acordo com as políticas das linhas que operacionalizamos.
- Verificar no menu ANEXOS CLIENTE quais documentos o cliente anexou à proposta.

3º PASSO - Entrar em contato com o cliente.

- Entrar em contato com o cliente para validar as informações dos passos acima.
- Explicar ao cliente as condições que ele terá para o financiamento proposto.
- Acessar a planilha RESPOSTAS SISTEMA para obter as mensagens padronizadas.

4º PASSO - Resultado da Análise do Enquadramento

- Caso proposta precise retornar ao cliente para correção/complementação
- Orientar o cliente por telefone sobre os ajustes necessário.
- Obter na planilha de RESPOSTAS SISTEMA as mensagens padronizadas sobre os ajustes e copiá-las para o campo MENSAGENS e no LIVRETO de mensagens internas.
- Assinalar no campo ENQUADRAMENTO INICIAL qual a linha será escolhida para o cliente. Quando a proposta retornar do cliente este campo pode ser alterado para outra linha conforme necessidade.
- Clicar em RETORNAR À ETAPA ANTERIOR (FASE 1) para que a proposta retorne para a fase INICIADO CLIENTE.
- Caso a proposta seja enquadrada.
- Orientar o cliente por telefone sobre os ajustes necessário.
- Obter na planilha de RESPOSTAS SISTEMA as mensagens padronizadas sobre os ajustes e copiá-las para o campo MENSAGENS e no LIVRETO de mensagens internas.
- Assinalar no campo ENQUADRAMENTO INICIAL qual a linha será escolhida para o cliente.
- Clicar em AVANÇAR para que a proposta siga para a próxima etapa.
